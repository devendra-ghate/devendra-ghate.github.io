Future work areas

  1. Evolutionary algorithm for multi-objective trajectory optimisation for an air-taxi using GIS data
    Air taxis are fast becoming a real proposition. Financially viable deployments will happen in dense urban areas with a skyline filled with high rises. At the same time, geospatial data is becoming available with increasing accuracy and ease. This data can be used to map an optimal trajectory for an air-taxi. A trajectory can minimise energy consumption, time or elevation. The last objective is for air-taxis used for tourist purposes. These multiple objectives make it a multi-objective optimisation. Population based algorithms are a natural choice to solve this multi-objective optimisation problem with discontinuous constraints.
    Requird skillls
    (Introduction of optimisation, familiarity with C/C++, interest in learning the usage of GIS software)

  1. Application of Machine Learning for Aerospace Applications
    Machine Learning libraries like DeepXDE and DifferentialEquations.jl can solve ODEs and PDEs. Advantages of using Neural Network based XDE solvers are meshless solution methodology, single interface for uncertainty propagration, robust PDE solutions etc. Trajectory optimisation, parameter estimation, hypersonic flow parameter estimation, solution of fluid mechanics equations etc. are some of the possible areas of application of these methods. This project aims to learn about, explore and report the possible Aerospace applications for this technology.
    Requird skillls
    (Familiarity with Python/Julia, interest in applied mathematics, introduction to machine learning)
