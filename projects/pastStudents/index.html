Neelappagouda V Hiregoudar
MTech, Aerospace Engineering, 2020
(Presently working as )
Multimodal optimization of NACA0012 wing with diffential evolution based parallel niching algorithm and reduced-order modeling technique
(Thesis)
\(\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \)
A test problem for multi-modal aerodynamics optimisation (Problem 6 from Aerodynamic design optimisaiton discussion group (ADODG)) is solved using a parallel niching differential evolution algorithm. Principal Component Analysis (PCA) is used for reduced order modelling on top of the Free Form Deformation (FFD) based geometric parameterisation. The entire project was implemented using Python, SU2 and Pointwise™.
Somayajula Venkata Janakiram
BTech, Aerospace Engineering, 2020
(Presently working as Scientist, System Reliability - Control Systems and Components group, LPSC, DOS, Thiruvananthapuram, India)
Optimal scheduling strategies for multiple agile Earth observation satellites (co-supervision with Dr. B S Girish, IIST)
(Thesis)
\(\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \)
This project developed population based iterated local search (P-ILS) and hybrid particle swarm optimisation local search (PSO-LS) algorithms to generate an optimal schedule for imaging requests coming from users for a constellation of imaging satellites.

Partha Ajit Surve
BTech, Aerospace Engineering, 2018
(Presently working as Scientist, Aero & Mission, STS, VSSC, DOS, Thiruvananthapuram, India (email))
A multi-fidelity aeroelastic optimisation of a wing using Co-Kriging (co-supervised with Dr. Palaniappan Ramu, IITM)
(Thesis)
\(\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \)
Aeroelastic wing optimisation problem is posed using co-Kriging multi-fidelity surrogate model. Resulting problem statement (with 10 continuous and 1 discrete design variable) is solved by implementing Multidisciplinary feasible (MDF) architecture using Ansys™ tools.
