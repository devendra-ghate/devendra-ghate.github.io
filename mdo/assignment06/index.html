Problem Statement

We wish to compare the rate of convergence of algorithms that we understand (from lectures and theory) and the algorithms that are implemented in commercial optimisation libraries. We will use this matlab implementation of Steepest Descent algorithm with Armijo's stepsize calculation[1].

Matlab's fminunc function has following algorithms:

  • quasi-Newton, and

  • trust-region.

We will use 2-dimensional Rosenbrock function as a test function. Plot the convergence of Rosenbrock function starting from the initial guess of \(\bar{x}^{(0)} = [1.5, 2.5]^T\).[2] Using these plots, estimate the convergence ratio (\(\beta\)) and the rate of convergence (\(r\)) for these algorithms. Set \(\left\lVert \Delta f \right\rVert_{2} \le 10^{-10}\).

We expect quasi-Newton and trust-region algorithms to converge faster that the plain vanilla implementation of steepest descent.
