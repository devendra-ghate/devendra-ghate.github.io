General Introduction

Genetic algorithms mimic evolution process.

  • DNA (Adenine, Thymine, Guanine, Cytosine) - chromosomes - genes -

  • Genotype-Phenotype

  • Survival of the fittest (natural selection)

Genetic algorithm is defined by following operations:

  1. Definition of the fitness function

  2. Definition of the encoding scheme for design variables

  3. Selection of population size

  4. Selection of the three main operators:

    • Reproduction (selection)

    • Crossover

    • mutation

  5. Stoppage criteria

We will take a closer look at all these.

Fitness Function

Objective function itself is set as the fitness function for the maximisation problems. However, for minimisation problems,

\[\mathcal{F}(x) = \frac{1}{1 + f(x)}\]

can be set as the fitness function. Note that \(\displaystyle \max_{x \in \mathbb{R}^n} \mathcal{F}(x)\) and \(\displaystyle \min_{x \in \mathbb{R}^n} f(x)\) have the same minima. This transformation does not alter the location of the minimum, but converts a minimization problem to an equivalent maximisation problem.

Question
If I define \(\mathcal{F}(x) = 1/f(x)\), then do the resulting problems have the optima at the same point? If yes, then why not use this transformation? If no, why do they have different optima?
Ans:

They have optima at the same point. But since we have assumed that \(f(x) \ge 0\), \(\mathcal{F}(x)\) is not always defined. Also, assuming \(f(x) \in [0, 1]\), then the range of \(\mathcal{F}(x)\) with this definition is \(\mathcal{F}(x) \in [1, \infty)\) as opposed to \(\mathcal{F}(x) \in [0, 0.5 ]\) as in the earlier transformation.

Reproduction Operator

Reproduction selects good strings in a population and forms a mating pool. There are many reproduction operators in GA literature, but the essential idea in all of them is that above-average strings are picked from the current population and their multiple copies are inserted in the mating pool in a probabilistic manner.

Proportionate reproduction operator (Roulette Wheel Operator)

  1. Calculate fitness of all the candidates in a population.

  2. Set probability of selecting each candidate to

\[p_i = \frac{\mathcal{F}_i}{\sum_{j=1}^{n} \mathcal{F}_j}\]

where \(n\) is the population size. This method is also known as Roulette Wheel selection. This can be implemented by sampling uniformly from \([0, 1]\) and selecting the string corresponding to that cummulative probability.
