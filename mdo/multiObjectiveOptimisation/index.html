Consider the following optimisation problem:

\[ \begin{equation*} \begin{aligned} & \underset{x}{\text{minimize}} & & length(x) = 100 x,~weight(x) = 100 x^2 \\ & \text{subject to} & & 0 \leq x \leq 1 \\ \end{aligned} \end{equation*} \] 
using Optim, PyPlot
#Design space
x = linspace(0,1,20)
#Functions to optimise
length(x) = 100*x; weight(x) = 100*x^2
#Plotting
len = map(length,x); w = map(weight,x)
scatter(len, w)
xlabel("length",fontsize=16); ylabel("weight",fontsize=16)

Now, lets consider the following optimisation problem:

\[ \begin{equation*} \begin{aligned} & \underset{x}{\text{minimize}} & & weight(x) = 100 x^2,~drag(x) = (x-2)^2 \\ & \text{subject to} & & 0 \leq x \leq 1 \\ \end{aligned} \end{equation*} \] 
#Functions to optimise
drag(x) = (x-2)^2
dr = map(drag,x)
#Plotting
scatter(w, dr); xlabel("weight",fontsize=16); ylabel("drag",fontsize=16)

Question: Which is the optimum point if we want to minimise \(f_2\) and \(f_3\) simulatneously ?

Need to define a strategy to pick one of the points from this Pareto front.

A point \(x*\) in the feasible design space \(S\) is called Pareto optimal if there is no other point \(x\) in the set \(S\) that reduces at least one objective function without increasing another one.

We need to formulate an objective function in such a way that one or all of the points on the pareto optimal front are obtained.

Strategy 1: Weighted Sum method

\[ f(x) = \underset{i}{\sum}~{w_i\,f_i(\mathbf{x})} \] 
## Strategy 1: Weighted Sum method

# Weighted objective function
w1 = 0.5; w2 = 0.5;
#w1 = 0.1; w2 = 0.9;
f4(x) = w1*weight(x) + w2*drag(x)
res = optimize(f4, -1, 1)
scatter(w, dr); hold("on")
xlabel("weight",fontsize=16); ylabel("drag",fontsize=16);
plot(weight(res.minimizer[1]), drag(res.minimizer[1]),"sr")

The consituent objective functions have to be normalised for the correct working of the weighted sum method.

## Strategy 1: Weighted Sum method with normalised objective functions
weight1(x) = x^2; drag1(x) = ((x-2)^2 - 1)/3;
# Weighted objective function
w1 = 0.5; w2 = 0.5;
#w1 = 0.1; w2 = 0.9
#w1 = 0.9; w2 = 0.1
#w1 = 0.0; w2 = 1.0;
#w1 = 1.0; w2 = 0.0;
f4(x) = w1*weight1(x) + w2*drag1(x)
res = optimize(f4, 0, 1)
w = map(weight1,x); dr = map(drag1,x)
scatter(w, dr); hold("on");
xlabel("weight",fontsize=16); ylabel("drag",fontsize=16)
plot(weight1(res.minimizer[1]), drag1(res.minimizer[1]),"sr")

Strategy 2: Weighted Min-Max method

\[ f(x) = \underset{i}{\text{max}}~{w_i(f_i(\mathbf{x}) - f_i^0)} \]

where \(f_i^0\) is the minimum of the \(i\) objective function.

  • Provides complete Pareto optimal set

  • May provide non-Pareto poitns as well

  • Requires individual minimum of the objective functions \(f_i\)

## Strategy 2: Weighted Min-Max method
weight1(x) = x^2; drag1(x) = ((x-2)^2 - 1)/3;
# Weighted objective function
w1 = 0.5; w2 = 0.5;
#w1 = 0.1; w2 = 0.9
#w1 = 0.9; w2 = 0.1
#w1 = 0.0; w2 = 1.0;
#w1 = 1.0; w2 = 0.0;
f5(x) = max(w1*(weight1(x) - 0), w2*(drag1(x) - 0))
res = optimize(f5, 0, 1)
w = map(weight1,x); dr = map(drag1,x)
scatter(w, dr); hold("on")
xlabel("weight",fontsize=16); ylabel("drag",fontsize=16)
plot(weight1(res.minimizer[1]), drag1(res.minimizer[1]),"sr")

Strategy 3: Weighted global criterion method

\[ f(\mathbf{x}) = \big\{\underset{i}{\sum}[w_i(f_i(\mathbf{x}) - f_i^0)]^p\big\}^{1/p} \]

where, \( p \in [1,\infty]\).

Exercise:

  1. Can you calculate the value of \(p\) so that strategy 3 reduces to strategy 1 & 2.

  2. What does the objective function \(f\) represents for \(p = 2\) ?
