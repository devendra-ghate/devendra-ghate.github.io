Line Search Methods

Consider the following unconstrained optimisation problem statement.

\[ \min_{\bar{x} \in \Re^n} f(\bar{x}) \]

where \(f(\bar{x})\) is a sufficiently smooth nonlinear function. The most obvious way of calculating the minima to solve the set of nonlinear equations arising from the necessary condition of optimality.

\[\nabla f(\bar{x}) = 0\]

This is only only possible if \(f(\bar{x})\) is an explicit function of \(\bar{x}\). When the explicit form of the objective function is not known, then necessary conditions of optimality cannot be used to find the minima. This is the case for most of the engineering problems.
