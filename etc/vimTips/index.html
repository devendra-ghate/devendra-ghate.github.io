Problem with freezing

Ctrl-s freezes vim. Use Ctrl-q to unfreeze it. When the tmux prefix of Ctrl-a is used before Ctrl-s then it doesn't freeze.

Path Variable

The path is set in the .vimrc file. ~/work/grok is added to it recursively using ** convention. Hence, use find filename<tab> to open all the markdown files.

Digraphs Superscripts/Subscripts

:digraphs will list all the digraphs. To insert a digraph, press C-k followed by the keycode for a digraph.

ActionResult
C-k e :ë
C-k e "ë
C-k 2 Sa²
C-k 2 sa₂

Subscripts and superscripts can be used in Julia in variable names.

Source: Vim templates

A templates directory (.vimrc/skeleton/) has been added and a line added to vimrc. The directory has template.extension files for various extensions and whenever a file with that extension is created, the template is added.

CTags

  1. Install exuberant ctags.

  2. Run ctags -R in the directory with the project. -R option will consider all the directories in a recurring fashion.

  3. Ctrl-] to go to the function.

  4. Ctrl-W Ctrl-] to open the function definition in a split window.

  5. Ctrl-T to go back.

  6. gCtrl-] or :sts to list all the instances of the matching tags (if any).
