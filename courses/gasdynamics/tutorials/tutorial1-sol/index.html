Tutorial 1 Solution

Problem 1

# Constants
γ = 1.4
Rᵤ = 8.3145 #Universal gas constant J/mole-K 

# Input
Mₒ = 32*10^-3 # molar mass of Oxygen in kg
T₁ = 25 + 273 # K
T₂ = 125 + 273 # K

# Calculations
R = Rᵤ/Mₒ #Mass specific gas constant for Oxygen J/kg-K 
cₚ = γ*R/(γ-1) #J/kg-K
cᵥ = cₚ - R  #J/kg-K
Δe = cᵥ*(T₂ - T₁)
Δh = cₚ*(T₂ - T₁)

Problem 2

# Constants
γ = 1.4
Rᵤ = 8.3145 #Universal gas constant J/mole-K

# Input
M = 28.97*10^-3 # molar mass of air in kg
p₁ = 101*1000 # Pascal (N/m^2)
p₂ = 1100*1000 # Pascal
T₁ = 310 # K

# Calculations
R = Rᵤ/M # J/kg-K
cₚ = γ*R/(γ-1) # J/kg-K
cᵥ = cₚ - R # J/kg-K
v₁ = p₁/R/T₁ # m/s
v₂ = (p₁/p₂)^(1/1.32) * v₁ # m/s
Δs = cᵥ*log(p₂/p₁) - cₚ*log(v₁/v₂) # J/kg-K

# Constants
γ = 1.4
Rᵤ = 8.3145 #Universal gas constant J/mole-K

# Input
Mₐ = 28.97*10^-3 # molar mass of air in kg
p₁ = 6.5*101325 # Pascal (N/m^2)
p₂ = 1*101325 # Pascal
T₁ = 440 # K
M₁ = 0.5 # Dimensionless
A₁ = 160 * 10^-4 # m^2

# Calculations
R = Rᵤ/Mₐ # J/kg-K

From isentropic table \(p_1/p_0 = 0.84\). From this I get the

\[p_2/p_0 = (p_2/p_1)(p_1/p_0) = (1/6.5)*0.84 = 0.129\]

Then again from the tables I get \(M_2 = 2\).

Lets try to do the actual calculations.

p₀ = p₁*(1 + (γ -1)/2* M₁^2)^(γ/(γ - 1))
p₁/p₀

This is a good enough match for the value we got from the tables. Now, I will calculate \(M_2\)"

M₂ = ( ( (p₀/p₂)^((γ - 1)/γ) - 1 ) * 2/(γ - 1) )^0.5

This is very close to 2. Hence, we have got the solution with a fraction of the effort without sacrificing much accuracy.

Now, from tables, \(A^*/A_2 = 0.6\)
